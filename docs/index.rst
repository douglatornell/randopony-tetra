RandoPony Documentation
=======================

This is the documentation for the RandoPony-tetra project.

RandoPony-tetra a re-implementation of RandoPony,
the BC Randonneurs Cycling Club online event registration system.
It uses:

* Python 3
* Pyramid for the web application framework
* SQLAlchemy for the database interface layer
* Bootstrap for responsive layout in the browser

The app is presently under development for the 2013 season.
